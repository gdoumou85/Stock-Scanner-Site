# Stock Opportunity Scanner Site

## Project Overview
An automated stock market analysis website with daily scans, curated insights, and data-driven commentary.

## Core Principles
- **Transparency**: This is AI-assisted analysis, not pretending to be something else
- **Value**: Original commentary on market data, not just scraped content
- **Automation**: I (the AI assistant) manage daily updates, scans, and content
- **Legitimacy**: Follow AdSense policies — original analysis on data

## Structure
```
stock-scanner-site/
├── content/          # Daily analysis posts (markdown)
├── templates/        # HTML templates for site generation
├── assets/           # CSS, JS, images
├── data/             # Scanned data, price history, watchlists
├── scripts/          # Automation scripts
└── site/             # Generated static site (GitHub Pages ready)
```

## Content Strategy
1. **Daily Market Snapshot** — Brief summary of market action
2. **Scanner Results** — Stocks hitting technical/fundamental criteria
3. **Deep Dives** — Individual stock analysis when warranted
4. **Watchlist Updates** — Tracking specific tickers over time

## Tech Stack
- Static site generator: Jekyll or plain HTML
- Hosting: GitHub Pages (free)
- Domain: Your own or github.io subdomain
- Updates: Automated via cron jobs I manage
